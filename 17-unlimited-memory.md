# Unlimited Memory — Kevin Horsley

## Overview

Kevin Horsley is a world memory champion who broke memory records after a brain injury left him struggling. His book teaches the same techniques used by all memory athletes — not natural talent, but trained technique. Simple, practical, immediately applicable.

## Core Framework

### The Four Laws of Memory

1. **Everything is in your brain** — You already know everything you need to know. You just haven't connected it properly yet.
2. **Memories are made by connecting new information to existing knowledge** — Everything meaningful you know is a connection.
3. **Your brain has a natural filing system** — Use location-based memory (memory palace) to organize.
4. **Visual, emotional, and physical memories are strongest** — Bizarre, vivid, sensory-rich images stick.

### The Master Key: Memory Palace

Same technique as Memory Craft and Moonwalking with Einstein:

1. Build a mental "palace" — a familiar place you know in detail
2. Place vivid, bizarre mental images of what you want to remember
3. Walk through the palace in your mind — images trigger recall

The more exaggerated, physical, sexual, or shocking the image — the more unforgettable.

### The Loci System (Method of Locations)

Create a mental journey through specific locations:

- Your home (front door → hallway → living room → kitchen → bedroom)
- A route you walk daily
- Any familiar spatial sequence

Assign one item per location. When you walk the route mentally, the location triggers the memory.

### Remembering Numbers

The Major System (from The Memory Book) — convert numbers to consonant sounds:

0=S/Z, 1=T/D, 2=N, 3=M, 4=R, 5=L, 6=J/CH/SH, 7=K/G, 8=F/V, 9=P/B

Create words from the sounds → associate with the concept.

Example: 314 = MRP = "MaRPi" → picture a fish with maracas → use in a memory palace.

### Remembering Names

1. **Hear it clearly** — ask if needed
2. **Picture the name** — does "Foster" look like a foster dog? A fence?
3. **Associate with a visual** — picture Mr. Foster standing with a foster dog, throwing a ball
4. **Review later** — use spaced repetition to cement

### The SPEAK System (for remembering lists)

S — State what you want to remember clearly
P — Picture it with a vivid, bizarre image
E — Embed it in a location
A — Associate the location to the next item
K — Keep reviewing with spaced repetition

### Why Cramming Fails

Cramming floods the brain with information but doesn't create strong neural pathways. The information is in short-term memory — it evaporates. The only way to transfer to long-term memory is: **understanding + connection + spaced repetition**.

### Focus Determines Success

Horsley's own story: after a brain injury, he was told he would never excel again. He chose to focus completely on memory training — and became a world champion. What you focus on expands.

## For Founders

- Remembering client names, details, conversations — a competitive advantage
- The Major System for remembering numbers: prices, dates, metrics
- Using memory palaces for remembering frameworks, strategies, competitive intel
- Focus matters: where you put your attention determines what you become exceptional at

## One-Line Takeaway

> Your memory isn't a recording device — it's a connection engine. Use vivid images, locations, and deliberate technique to unlock unlimited recall.
