# Moonwalking With Einstein — Joshua Foer

## Overview

Joshua Foer wanted to understand how memory works — and accidentally became a USA Memory Championship winner in one year. This book is his journey into the world of memory athletes, ancient techniques, and the nature of memory itself.

## The Forgetting Curve

Hermann Ebbinghaus discovered that we forget roughly 50% of new information within an hour, 70% within 24 hours. Memory is inherently fragile — forgetting is the default, not the exception.

## Memory Athletes Are Made, Not Born

The world memory champions aren't geniuses with photographic memories — they're people who trained their memory using ancient techniques. With practice, anyone can dramatically improve recall.

## The Memory Palace

The core technique: visualize a familiar place (your home), then place vivid, bizarre mental images of things you want to remember along the route. To recall, mentally walk through the place.

- Ancient Romans used this technique (method of loci)
- Cicero used it to memorize speeches
- The more bizarre and vivid the image, the more memorable
- Physical, sensory, sexual, shocking images stick better than bland ones

## Key Principles

### Attention
You cannot remember what you never paid attention to. Attention is the gateway to memory.

### Emotion
Emotionally charged events are remembered better. This is why shocking, funny, or embarrassing moments are memorable — they're emotionally arousing.

### Visualization
The more vividly you can visualize something, the better you remember it. Bizarre, exaggerated, multisensory images are best.

### Association
New information is remembered by connecting it to what you already know. The memory palace is essentially a web of associations.

## The Myth of Photographic Memory

True photographic (eidetic) memory doesn't exist in adults. What memory athletes do is not photographic — it's technique. Anyone can learn the techniques.

## Practical Applications

- Use a memory palace for lists, speeches, historical dates, foreign language vocabulary
- The technique works — but it requires practice and mental effort
- The effort itself may be what makes it stick

## For Founders

- Memory palaces useful for memorizing business frameworks, pitch decks, competitive landscape
- More importantly: understand how attention and emotion drive what you remember — this applies to what customers remember about your product

## One-Line Takeaway

> Memory is a skill, not a talent — with the right techniques, extraordinary recall is available to anyone.
