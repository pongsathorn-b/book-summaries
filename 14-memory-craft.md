# 14 — Memory Craft: Improve Your Memory with the World's Best Memory Techniques

**By Lynne Kelly (2019)**

---

## Overview

*Memory Craft* is the result of Lynne Kelly's personal journey from someone with no exceptional memory to a gold medalist at the World Memory Championships. Kelly is a senior lecturer in education at La Trobe University (Australia) and a researcher who approached memory techniques as a cognitive scientist would — rigorously, experimentally, and with an eye toward what actually works for ordinary people. The book systematically dismantles the myth that good memory is a innate talent, replacing it with the understanding that expert memory is a **learnable skill** built on specific, trainable techniques. Kelly argues that the memory palace (method of loci) and spatial visualization techniques used by ancient Greek and Roman orators, medieval scholars, and modern memory athletes are not party tricks — they are powerful cognitive tools that anyone can master. The book is particularly notable for connecting modern memory research to Indigenous knowledge systems and oral traditions, showing how these same techniques underpin the most sophisticated memory systems ever developed.

## Core Concepts

### Memory Is a Trainable Skill, Not a Talent
The central thesis of *Memory Craft* is that exceptional memory is not a product of innate intelligence or genetic gift — it is the result of specific techniques, practiced deliberately. The brain's hippocampus, critical for memory formation, is highly plastic throughout life. Studies of London taxi drivers (increased hippocampal volume) and memory athletes (who use mnemonic encoding strategies) demonstrate that memory capacity changes in response to use. This is empowering: anyone can improve their memory substantially with the right approach.

### The Memory Palace (Method of Loci)
The memory palace is a technique in which you mentally place items you want to remember along a familiar spatial route or inside a familiar building. To recall them, you mentally walk the route and "see" the items in each location. The technique exploits the brain's exceptional spatial and episodic memory systems — humans are extraordinarily good at remembering places and routes because these abilities were critical for survival (navigation, food storage, predator avoidance). By converting abstract information into vivid, unusual, often exaggerated imagery and placing that imagery in a spatial context, the memory palace allows recall of sequences, lists, and structured information that would otherwise require constant reference to notes.

### The Importance of Vivid, Unusual, Often Gross Imagery
For memory palace encoding to be effective, images must be **vivid, interactive, and emotionally engaging** — even shocking or gross. A flying purple elephant interacting with your list item will be remembered far better than a plain word. Unusual, exaggerated, sexually suggestive, or surprising imagery triggers the brain's "novelty detectors" and creates stronger neural traces. Kelly calls this **"make it weird"** — the stranger the image, the more memorable.

### Encoding Depth and Meaning
Shallow processing (reading, rereading, passive review) produces shallow, fragile memory traces. Deep processing — understanding *why* something matters, how it connects to existing knowledge, and what it means — creates richer, more durable memories. Kelly combines mnemonic technique (which provides structural encoding) with genuine intellectual engagement (which provides semantic depth). Both are necessary for truly durable memory.

### Spaced Repetition (Review at Increasing Intervals)
Kelly incorporates the Ebbinghaus forgetting curve research into her practice. Reviewing material at spaced intervals is dramatically more effective than massed practice (cramming). The memory palace itself is a spacing tool — you revisit the same palace over time, each visit strengthening the traces. Modern spaced repetition software (Anki) can be integrated with memory palace practice for maximal efficiency.

### The Chunker: Organizing Information Before Memorizing
Before placing information in a memory palace, it must be organized. Chunking — grouping items into meaningful sets of 5–9 (Miller's magic number) — makes encoding more efficient. The order itself can be encoded using a phonetic number system (the Major System) or a keyword system, which Kelly explains in detail.

### Indigenous Knowledge Systems and Oral Tradition
One of the most distinctive aspects of *Memory Craft* is Kelly's research into how Indigenous cultures, particularly Australian Aboriginal songlines and Native American knowledge transmission, used these same memory techniques on a civilizational scale. Indigenous knowledge keepers developed extraordinarily complex memory systems — entire encyclopaedias encoded in song, dance, and visual art — that preserved vast bodies of knowledge across generations without writing. These systems demonstrate the power of the memory palace and spatial encoding at the highest level, and Kelly draws important lessons from them about embodied, multi-sensory, emotionally rich learning.

## Key Lessons

1. **Exceptional memory is a skill, not a gift.** With the right techniques and deliberate practice, anyone can dramatically improve their memory.
2. **The memory palace is the cornerstone technique.** Use familiar spatial routes as a filing system for information you need to recall in sequence.
3. **Make images vivid and weird.** The more unusual, exaggerated, and emotionally charged your encoding images, the more indelible the memory trace.
4. **Understand before you memorize.** Deep semantic understanding combined with mnemonic structure creates far more durable memory than either alone.
5. **Space your reviews.** Each review of material at increasing intervals strengthens the trace exponentially.
6. **Your body can remember.** Embodied, multi-sensory learning — involving movement, sound, smell, touch — creates richer memory traces than passive reading.
7. **Memory champions use systems, not talent.** The people who win memory competitions are using techniques derived from ancient Greek and Roman rhetorical traditions — techniques that are completely learnable.
8. **Teach to remember.** Having to teach someone else forces deep processing and identification of gaps — one of the most powerful memory techniques available.

## Practical Applications

- **For students**: Build memory palaces for historical dates, scientific classifications, foreign language vocabulary, and process flows. Use spaced repetition (Anki) in conjunction with palaces. Review palaces at increasing intervals.
- **For professionals**: Memorize speeches without notes (build a palace for each section), learn names and faces systematically, retain procedural knowledge more durably.
- **For anyone learning a skill**: Use the memory palace to encode the steps of a complex process, the sequence of a musical piece, or the key principles of a new field.
- **For retaining books and reading**: Build palaces for key concepts from books, walking through the "building" and seeing each concept in its location. This is vastly more effective than highlighting.
- **For creative work**: Memory palaces can store creative combinations — bring together unexpected "items" from different palaces to generate novel connections and ideas.
- **To build lasting knowledge**: Combine memory palace technique with genuine conceptual understanding — ask why things work, how they connect, what examples illustrate them.

## One-Line Takeaway

> Lynne Kelly proves that exceptional memory is a learnable art built on ancient techniques — primarily the memory palace — made systematic and accessible, showing that anyone can transform their memory by transforming how they encode information.
