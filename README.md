## 📁 Format

Each summary follows this structure:
1. **Overview** — what the book is about
2. **Core Concepts** — 4-6 main ideas
3. **Key Lessons** — 8-10 numbered takeaways
4. **Practical Applications** — how to apply it
5. **Controversy/Criticism** — common objections
6. **Related Books** — Obsidian wikilinks to connected summaries
7. **One-Line Takeaway** — the distillation
